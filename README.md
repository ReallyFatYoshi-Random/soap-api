Soap API Server created in PHP.

## Run project locally
Open your terminal.

1. Clone repository
    ```sh
    git clone https://github.com/ReallyFatYoshi-Random/soap-api.git
    ```

2. Open repository
    ```sh
    cd soap-api
    ```

3. Install dependancies
    ```sh
    composer install
    ```

4. Run the application
    ```sh
    composer run serve
    ```
